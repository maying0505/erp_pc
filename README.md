德智金服-后台管理系统
##启动
npm run dev
##打包
npm run build
## 技术栈

- react
- antd
- webpack
- es6(babel)

## 兼容性

原则上支持 IE9+ 及现代浏览器

## 环境

- node 4 LTS 版本
- npm 建议 3+
- webpack 3

> npm 建议使用cnpm, 通过设置 alias 的方式;(在私有npm还没搭建起来的时候)

## 开发

```bash

    cnpm install

    npm run dev

```
### 设置 api 代理

可在 webpack.dev.config.js 里面的 devServer 配置项设置 api 代理
